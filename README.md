# Data Analyst Salary Analysis

This project analyzes Data Analyst salary trends across experience levels, employment types, and company locations using structured SQL analysis.

The objective of this analysis is to identify compensation patterns, understand salary progression by seniority, and highlight growth gaps between career levels to support data-driven insights.
